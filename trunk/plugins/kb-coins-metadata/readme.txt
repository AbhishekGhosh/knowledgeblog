KBlog COinS Metadata Exposer is a plugin to make a Knowledgeblog (http://knowledgeblog.org) readable by COinS interpreters.

== Description ==

Adds COinS &lt;span&gt; tags to the end of the_content. The tag contains information about the post title, author(s), categories etc. The provision of this tag makes the blog posts readable by interpreters able to understand the COinS format, such as Zotero and Mendeley.

== Installation ==

1. Unzip the downloaded .zip archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
