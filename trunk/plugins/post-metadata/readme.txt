A tool for adding metadata to Knowledgeblog articles, or scholarly articles written in Wordpress. Adds meta-tags to post headers, and provides metadata from stable URLs.

== Description ==

Provides metadata for scholarly works. Supported data includes author, title, date, doi etc.

== Installation ==

1. Unzip the downloaded .zip archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Using the plugin settings page, set which identifier you want processed as the default (DOI or PMID).
